# 🩺 Medic Wellness

A mini healthcare support web app built with React and Claude AI.

## Features
- Patient support registration form
- Volunteer registration form
- AI-powered chatbot for FAQs
- AI auto-response on form submission

## Tech Stack
- React + Vite
- Anthropic Claude API

## Live Demo
[View Live] https://medic-welness-juk1.vercel.app

## How to Run
```bash
npm install
npm run dev
```
